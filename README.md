# Ночной рейс

Самодостаточная браузерная survival-игра. Визуальные ассеты сгенерированы, HUD и логика игры рисуются нативно на canvas.

## Запуск локально

```bash
cd /Users/nechepkasov/night-fare
npm run dev
```

Открой `http://localhost:5173`.

## Как открыть доступ из интернета

Игра статическая, поэтому локальный сервер может слушать все сетевые интерфейсы:

```bash
cd /Users/nechepkasov/night-fare
python3 -m http.server 5173 --bind 0.0.0.0
```

После этого открой TCP-порт `5173` одним из способов:

- Port forwarding на роутере: внешний TCP `5173` направить на LAN IP этого Mac и TCP `5173`.
- Туннель: `ngrok http 5173` или `cloudflared tunnel --url http://localhost:5173`.
- VPS/reverse proxy: скопировать папку на статический хост или отдать через Nginx.

Не открывай порт, если не доверяешь сети. Для публичного доступа безопаснее статический хост или reverse proxy с HTTPS, чем голый проброс домашнего порта.

## Управление

- Мышь или touch: прицел.
- Удерживать ЛКМ или Пробел: рывок.
- A/D или стрелки: уклон.
- C: восстановить дыхание.
- P или Esc: пауза.
- Enter: рестарт после победы/поражения.
- На тач-устройствах под игрой появляются кнопки «УКЛОН» и «ВДОХ».

## Правила

Не тяни верёвку постоянно: ранний рывок снижает здоровье, панику и сбрасывает комбо. Жди короткое окно, когда водитель теряет контроль, держи прицел на подсвеченной точке у петли и следи за выносливостью.

В фазе «ПОВОРОТ» водитель тянется к блокировке — уклонись в **подсвеченную стрелкой сторону** (влево или вправо), иначе получишь урон. Чистый уклон даёт бонус к счёту и успокаивает.

Чем выше паника, тем быстрее и громче бьётся сердце. Победа даёт счёт за скорость, здоровье и комбо; лучший результат сохраняется в браузере.
